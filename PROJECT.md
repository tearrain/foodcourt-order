# 🍜 食阁扫码点单系统 - 项目文档

**创建时间：** 2026-02-15
**状态：** MVP 规划中

---

## 一、业务需求

**核心场景：**
- H5 页面，用户扫码后打开
- 展示食阁所有档口美食集合
- 支持按美食选择，或按档口选择
- 跨档口聚合购物车
- 一次支付，后台自动完成订单商品统计和分账结算
- 推荐美食功能，提高转化率

**目标用户：**
- C端：消费者（扫码点餐）
- B端：食阁管理者、档口经营者

---

## 二、技术选型

| 层级 | 推荐方案 |
|------|----------|
| H5前端 | Vue3 + Vite + TailwindCSS |
| 后端 | Cloudflare Workers |
| 数据库 | PostgreSQL (D1 / Supabase) |
| 支付 | 聚合支付 + 分账 |
| 部署 | Cloudflare Pages + Workers |

---

## 三、功能规划

### MVP（核心闭环）
1. 扫码入口（H5）
2. 食阁/档口/菜品展示
3. 聚合购物车
4. 统一支付
5. 订单分发

### Phase 2（增强功能）
1. 推荐系统
2. 评价系统
3. 优惠系统

### Phase 3（生态扩展）
1. 档口SaaS
2. 数据分析
3. 会员体系
